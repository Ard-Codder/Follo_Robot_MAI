Чтобы связать Raspberry Pi 4 и Raspberry Pi Pico, мы можем использовать последовательный
    интерфейс (UART) через GPIO-пины. Вот шаги, которые нам нужно выполнить:

1. Подключение:
    Подключите GND (PIN 6 на Raspberry Pi 4 и PIN 34 на Raspberry Pi Pico)
    Подключите TX (PIN 8 на Raspberry Pi 4 и PIN 0 на Raspberry Pi Pico)
    Подключите RX (PIN 10 на Raspberry Pi 4 и PIN 1 на Raspberry Pi Pico)
    Важно отметить, что RX на Raspberry Pi 4 должен быть подключен к TX на Raspberry Pi Pico и наоборот.

2. Настройка Raspberry Pi 4:
    Откройте терминал на Raspberry Pi 4.
    Установите необходимые пакеты, выполнив команду: sudo apt-get install minicom screen.
    Откройте файл конфигурации последовательного порта: sudo nano /boot/config.txt.
    Добавьте в конец файла следующие строки, чтобы отключить автоматическую консоль через последовательный порт:

    enable_uart=1
    dtoverlay=pi3-miniuart-bt

    Сохраните и закройте файл (CTRL+X, Y, Enter).
    Перезагрузите Raspberry Pi 4: sudo reboot.

3. Настройка Raspberry Pi Pico:
    Загрузите на Raspberry Pi Pico соответствующую программу (прошивку), которая будет использовать последовательный
    интерфейс (UART) для общения. Вы можете использовать MicroPython или C/C++ для написания программы.

4. Общение:
    После настройки обоих устройств, вы можете использовать программу minicom или screen на Raspberry Pi 4 для
    общения с Raspberry Pi Pico через последовательный интерфейс.

    Например, вы можете использовать команду screen /dev/ttyAMA0 115200 для запуска терминала, подключенного к
    последовательному порту со скоростью 115200 бод.

Теперь вы можете обмениваться данными между Raspberry Pi 4 и Raspberry Pi Pico через последовательный интерфейс.
